Sobre o Sistema

Este sistema utiliza o modelo CLIP (Contrastive Language-Image Pre-training) da OpenAI para encontrar imagens similares em um dataset.

Tecnologias utilizadas:

  • CLIP: Modelo de IA da OpenAI treinado em 400 milhões de pares imagem-texto
  • FAISS: Biblioteca para busca eficiente de similaridade
  • PyTorch: Framework de deep learning
  • Flask: Framework web em Python
  • SQLite: Banco de dados para persistência

O sistema converte imagens em vetores (embeddings) e busca as imagens mais próximas no espaço vetorial.
