Sobre o Projeto

Tecnologia

Este sistema utiliza o modelo CLIP da OpenAI para extrair características de imagens e encontrar similares usando embeddings vetoriais.

Dataset

O sistema foi treinado no dataset E-PillID, contendo imagens de diferentes tipos de comprimidos e medicamentos.

Funcionamento

1. A imagem enviada é convertida em um vetor de características (embedding)

2. O sistema busca no banco de dados os vetores mais próximos

3. As imagens correspondentes são retornadas ordenadas por similaridade
